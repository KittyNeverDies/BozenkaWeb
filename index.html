About Bozenka

Multi Purpose & cross platform bot, wrote on python, was made for training mine skills (kittyneverdies), right now is actively maintained. Named from English language Bozo, what means clown.

Features

  • Already Telegram realization
  • Community driven, licensed under GPL-v3 license
  • AI generation queue system
  • Gpt4Free, Gpt4All and FusionBrain API support
  • Async database using SQLAlchemy + PostgreSQL

Future

In future, we would like to realize something like bridge from social platforms, what allow users to unite their comminities. We would like to work on discord, matrix, vk and other messaging platforms.
